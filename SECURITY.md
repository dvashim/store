# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, report it through [GitHub Security Advisories](https://github.com/dvashim/store/security/advisories/new).

You should receive a response within 48 hours. Once confirmed, a fix will be released as soon as possible and you will be credited in the advisory (unless you prefer to remain anonymous).
